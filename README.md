## Env
JDK 1.8 + Spring MVC Boot

## IDE

Spring Tool Suit

## Run 

- git clone https://github.com/coolcode/pha.git
- load project by using Spring Tool Suit
- right click project -> Run As ->Spring Boot App
- browser http://localhost:8080
